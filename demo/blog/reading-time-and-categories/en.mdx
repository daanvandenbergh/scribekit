## Reading time is precomputed

The `Blog` class estimates a whole-minute reading time from each post's body at parse time, so
the overview can show "N min read" without loading the full article.

## Categories drive the filters

List one or more `categories` in front-matter and the overview turns them into filter buttons -
unless every post shares a single category, in which case the bar is hidden.
