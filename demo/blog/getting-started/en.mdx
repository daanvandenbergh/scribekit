## What this is

This post lives at `demo/blog/getting-started/en.mdx`. The folder name is the slug, so it is
served at `/blog/getting-started`.

## How it works

The `Blog` class reads this folder and `<BlogOverview>` / `<BlogPage>` render it. You write
Markdown; the package handles the routing metadata, SEO JSON-LD, and the post sidebar.
